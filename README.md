# 📖 About The Project

Smart-Shop is a Flask-powered shopping platform designed for modern online shopping experiences.  
The system includes responsive navigation, product showcases, premium membership pages, return policies, and interactive UI components.

This project was built for learning full-stack web development and improving frontend/backend integration skills.

---

# ✨ Features

- Responsive navigation bar
- Homepage product banners
- Flash sale marquee section
- Premium membership page
- Warranty & return policy pages
- Dynamic Flask template rendering
- Organized static asset management
- Responsive CSS layout
- JavaScript interactive effects
